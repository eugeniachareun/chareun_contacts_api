## Programación de Aplicaciones Móviles III - Actividad Integradora 1

Proyecto de API REST para CRUD de contactos.

## Features
### Seguridad
*  Login con credenciales hardcodeadas
*  Uso de JWT
*  Protección de endpoints del controlador

### CRUD
* Modelo de contacto:
  * Datos: Nombre, apellido, teléfono y mail
* Lista de contactos en memoria
* Permite editar un contacto
* Permite agregar un contacto

### Desarrollo
* Swagger con endpoints documentados, incluye login
